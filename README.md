# node.js_bookstore
Node.js를 활용한 서점 사이트

#### - server: node.js express 사용
#### - front: ejs 사용

#### - lib: js 파일 모음 (서버 기능)
#### - views: ejs 파일 모음 (화면 구성)
#### - public/images: 사용되는 파일 및 이미지 모음
